# Bizflow – Inventory & GST Billing System

A complete web-based billing and inventory management system for small businesses.

---

## Features

- **Inventory** – Add/edit/delete products with name, price, quantity, unit, category
- **GST Billing** – Create invoices with automatic CGST + SGST calculation
- **Professional PDF** – Download invoices and reports as PDF
- **Parties** – Manage customers, view per-party ledger
- **Payments** – Track UPI and cash payments
- **Reports** – Stock report + Monthly sales report with PDF export
- **Login** – Secure JWT-based authentication

---

## Requirements

- Node.js v18 or higher
- MySQL 8.0 or higher (MySQL 9.x Community Server is also fine)
